# Brain学習アシスト Chrome拡張機能 — 引き継ぎ設計仕様書

## 1. 概要

**目的**: brain-market.com の長大な記事（例:『明鏡』は309セクション）をKindleのように快適に学習するためのChrome拡張機能。

**対象ドメイン**: `https://brain-market.com/*`

**ファイル構成**:
```
brain-reader/
├── manifest.json   (Manifest V3)
└── content.js      (単一ファイルで完結。CSSはJS内でstyle要素として注入)
```

---

## 2. ページ構造（重要）

Brain記事ページはVue.jsで構成されており、以下のセレクタで各要素を特定する。

```
本文エリア    : ._body_vrp2u_275  (フォールバック: [class*="_body_"], main)
見出し        : 本文エリア内の h2, h3, h4
見出しのid    : toc_1_0, toc_1_1, toc_9_3 ... の形式で付与済み
TOCラッパー   : ._toc_wrapper_vrp2u_478  (既存TOCは無視してよい)
```

**注意点**:
- Vue仮想スクロールのため、`scrollIntoView()` や `window.scrollTo()` で大きくジャンプするとページが白く消える場合がある（Brainの既知の挙動）
- 見出しへのアクション追加は `setInterval` で定期実行し、動的レンダリングに対応する
- ページ内の全h2/h3数は309個（『明鏡』の場合）

## 5. 機能詳細

### 5-1. 目次タブ
- 本文エリアの h2/h3/h4 を全件リスト表示
- h2 は太字・標準サイズ、h3 は左に18px字下げ、h4 は30px字下げ
- 各アイテム左端に既読チェックマーク（✓ / ·）、栞済みなら橙のドット
- クリック → `scrollIntoView({behavior:'smooth', block:'start'})` + 既読登録
- テキスト入力による絞り込み検索（`#br-search`）

### 5-2. 栞（ブックマーク）タブ
- 見出しにマウスオーバーすると🔖ボタンが出現（CSS opacity: 0→1）
- 🔖クリック → 追加 or（既存なら）削除のトグル
- 一覧表示: タイトル + 追加日 + 🗑削除ボタン
- アイテムクリック → 該当見出しへジャンプ

### 5-3. ハイライト・メモタブ
**ハイライト手順**:
1. テキストを選択
2. カラーポップアップが `selection.getRangeAt(0).getBoundingClientRect()` の直上に出現
3. 色ボタンクリック → `range.surroundContents(span)` でDOM内にspan挿入
4. 📝メモボタン → モーダルでメモテキストを入力してから適用

**ハイライトspan仕様**:
```html
<span class="br-highlight br-highlight-yellow" data-hl-id="hl_xxx" title="📝 メモ内容">
  ハイライトされたテキスト
</span>
```
- Alt+クリック → DOM上のspan除去 + データから削除
- `surroundContents()` が失敗する複雑な範囲はtoastでエラー表示

**一覧表示**: 新着順。テキスト(最大100文字) + メモ + セクション名 + 日付 + 削除ボタン

**エクスポート**: 「📄 Markdownでコピー」ボタンで全ハイライトを以下形式でクリップボードに書き出す:
```markdown
# Brain学習ハイライトまとめ
作成日: 2026/4/14

---

## 1. セクション名
> ハイライトテキスト

📝 メモ: メモ内容

## 2. ...
```

### 5-4. 統計タブ
- 学習進捗バー: `readSections.length / 全h2+h3数 * 100`
- 4枚のカード: 既読セクション数 / 全セクション数 / 栞数 / ハイライト数
- 「進捗リセット」ボタン: readSections と lastPosition をクリア（栞・HLは保持）
- 「全データ削除」ボタン: 全データをクリア（confirm確認あり）

### 5-5. 再開バナー
- ページ読み込み2秒後に表示
- 条件: `ST.lastPosition > 300`
- 内容: 最後に読んでいた見出しテキスト + 「再開 →」ボタン
- 再開ボタン: `window.scrollTo({top: ST.lastPosition, behavior: 'smooth'})`
- 8秒後に自動消去

### 5-6. 進捗バー・既読追跡
- scroll イベント（600msスロットル）で処理
- `scrollY / (scrollHeight - innerHeight) * 100` → 進捗バー幅に反映
- 画面上部30%に入った h2/h3 を自動的に既読登録

## 7. manifest.json

```json
{
  "manifest_version": 3,
  "name": "Brain学習アシスト",
  "version": "1.0",
  "description": "Brain記事をKindleのように読む：栞・ハイライト・メモ・進捗管理",
  "permissions": ["clipboardWrite"],
  "content_scripts": [
    {
      "matches": ["https://brain-market.com/*"],
      "js": ["content.js"],
      "run_at": "document_idle"
    }
  ]
}
```
※ `localStorage` はpermissions不要。`clipboardWrite` はエクスポート機能で使用。
