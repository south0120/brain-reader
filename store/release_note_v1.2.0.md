# Brain Reader v1.2.0 リリースノート

## 新機能

### 🔄 Chrome Sync対応
- 学習カレンダーのデータがChromeプロファイル経由で **端末間同期** されるようになりました
- 自宅PC・会社PC・ノートPCなど、異なるデバイスで学習状況を共有できます

### 🔥 Duolingo風ストリーク演出
- 連続学習日数を自動カウント
- 2日以上連続でトースト通知、5日以上でポップアップバナーが表示されます
- 7日・14日・30日突破でメッセージが変化

### 𝕏 Xシェア機能（3箇所）
- **ハイライトメモ** — 各ハイライトに「𝕏 ポスト」ボタンを追加。学びの気づきをXにシェア
- **学習カレンダー** — 連続日数をXにポスト「🔥 N日連続中！」
- **学習状況** — カレンダー下の「学習状況をXで共有する」ボタンで進捗率・連続日数・栞数・ハイライト数をまとめてポスト

### 🔗 紹介リンク（アフィリエイト）自動付与
- ヘルプタブからBrainの紹介URLを1回貼り付けるだけで設定完了
- 以降のXシェアすべてに紹介リンクが自動で含まれます
- 記事ごとにキャッシュされるので、次回以降は設定不要

## 改善
- ヘルプタブに「Xシェア・紹介リンク設定」セクションを追加
- ハイライト一覧のUIにアクションボタン行を追加

- **対象ブラウザ**: Google Chrome（Manifest V3）
