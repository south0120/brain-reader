# Brain Reader

Brain ([brain-market.com](https://brain-market.com/)) の長文記事を Kindle のように快適に読める Chrome 拡張機能。

**目次・栞・ハイライト・メモ・進捗管理・学習カレンダー** をサイドバーから一括操作。

## 主な機能

- 📋 **目次** — 全見出しを一覧表示、検索、現在位置の自動追従、既読チェック
- 🔖 **栞** — 見出しにマウスオーバーして1クリック登録、後でジャンプ
- 🖊 **ハイライト** — 4色のマーカー、メモ付き、複数段落対応
- 📊 **学習カレンダー** — 月ごとのヒートマップで学習日と既読セクション数を可視化、連続日数表示
- 📈 **進捗バー** — スクロール位置と既読率の自動追跡
- 🔄 **続きから再開** — 前回の閲覧位置をワンクリックで復帰
- 🎓 **チュートリアル** — 初回起動時にインタラクティブツアー
- 📄 **Markdown 書き出し** — ハイライト一覧をクリップボードへ

## インストール

### 1. リポジトリを取得

```bash
git clone https://github.com/south0120/brain-reader.git
```

または GitHub の **Code → Download ZIP** から取得して解凍。

### 2. Chrome 拡張機能として読み込む

1. Chrome で `chrome://extensions/` を開く
2. 右上の **デベロッパーモード** を ON
3. **パッケージ化されていない拡張機能を読み込む** → `brain-reader` フォルダを選択

### 3. Brain の記事ページを開く

[brain-market.com](https://brain-market.com/) の任意の記事ページを開くと、右端にブックアイコンが表示されます。クリックでサイドバーが開きます。

初回起動時は自動でチュートリアルが流れます。

## 操作チートシート

| 操作 | 方法 |
|------|------|
| サイドバーを開く/閉じる | 右端のブックアイコンをクリック |
| セクションへジャンプ | 目次タブでクリック |
| 栞をつける | 見出しにマウスを乗せて栞ボタンをクリック |
| ハイライト | テキストを3文字以上ドラッグ選択→色を選ぶ |
| メモ付きハイライト | 選択後、ペンアイコン→メモ入力→保存 |
| ハイライト削除 | `Alt` + クリック、または一覧の削除ボタン |
| 続きから読む | 再訪時に右端のバナーの「再開」をクリック |
| 使い方を見る | 「使い方」タブ |
| Markdown 書き出し | ハイライトタブの「Markdown でコピー」 |

## データの仕組み

- 全データは **ブラウザの `localStorage` にローカル保存**。サーバーへの送信は一切ありません。
- 記事ごとに独立したストレージキー（URL ハッシュベース）で管理されるため、データ混在しません。
- 学習アクティビティは記事ごとと全記事横断の両方を集計。

## 技術仕様

- Manifest V3 / Vanilla JavaScript（外部依存なし）
- 単一の `content.js` で完結
- Chromium 系ブラウザ（Chrome / Edge / Brave / Arc 等）対応

## 開発

```
brain-reader/
├── manifest.json   # 拡張機能定義
├── content.js      # 本体（CSSはJS内でstyle要素として注入）
├── icon16.png ~ icon128.png
└── 設計書.md        # 内部設計仕様
```

`content.js` を編集後、`chrome://extensions/` の更新ボタンで反映されます。
